- 問題: [349. Intersection of Two Arrays](https://leetcode.com/problems/intersection-of-two-arrays/description/)
- コメント集: []()
- 条件
  - 1 <= nums1.length, nums2.length <= 1000 
  - 0 <= nums1[i], nums2[i] <= 1000
- 方針1
  - 二重ループを回すと `10^6` になってしまうので一回ずつ回した方が良さそう
  - `nums1` を先に Set に追加する
  - `nums2` を次に Set に追加する
  - 上記で作成した2つのSetの積集合をとり、int[] に格納すれば良さそう
  - 確かSetの親クラスであるCollectionsにそんな感じのメソッドがあったはず
- 時間計算量: `O(N)`
- 空間計算量: `O(N)`

## 1st
```java
class Solution {
    // 本当は Optional<Int>を返したい
    public int[] intersection(int[] nums1, int[] nums2) {
        if (nums1.length == 0 || nums2.length == 0) {
            return null;
        }

        Set<Integer> num1Unique = new HashSet<>();
        for (int num : nums1) {
            num1Unique.add(num);
        }   

        Set<Integer> num2Unique = new HashSet<>();
        for (int num : nums2) {
            num2Unique.add(num);
        }

        num1Unique.積集合できるメソッド(num2Unique);

        int[] result = new int[num1Unique.size()];
        for (int i = 0; i < num1Unique.size(); i++) {
            result[i] = num1Unique.get(i);
        }

        return result;
    }
}
```

## 2nd
- 動くようにした
- Collectionsのメソッドは [retainAll()](https://docs.oracle.com/javase/jp/7/api/java/util/Collection.html#retainAll(java.util.Collection))
```java
class Solution {
    // 本当は Optional<Integer[]>を返したい
    public int[] intersection(int[] nums1, int[] nums2) {
        if (nums1.length == 0 || nums2.length == 0) {
            return null;
        }

        Set<Integer> num1Unique = new HashSet<>();
        for (int num : nums1) {
            num1Unique.add(num);
        }   

        Set<Integer> num2Unique = new HashSet<>();
        for (int num : nums2) {
            num2Unique.add(num);
        }

        num1Unique.retainAll(num2Unique);

        int[] result = new int[num1Unique.size()];
        int i = 0;
        for (int intersection : num1Unique) {
            result[i] = intersection;
            i++;
        }

        return result;
    }
}
```
