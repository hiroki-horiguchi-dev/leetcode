### 1st
- [問題](https://leetcode.com/problems/reverse-linked-list/description/)
- [コメント 206](https://docs.google.com/document/d/11HV35ADPo9QxJOpJQ24FcZvtvioli770WWdZZDaLOfg/mobilebasic#h.x5w37bodndgj)
- 方針3つ
  - `Iterative`
    - ポインタを用意し、ループを回しながらO(N)で ListNode を走査してポインタの張り替えを行う
    - 考えることが多いのがネックだと思う
      - 考えること
        - 次のノードがあるのか？と張り替え処理を同時に行わないといけない
        - 3つのポインタを気にしないといけない
  - `Recursion1` これが一番良い
    - 終点まで行ったら、一つずつ戻りながら処理する
  - `Recursion2`
    - 終点までいく過程で、処理してしまう (Iterative に近い方針)
  - `Stack`
    - Stack にためていき、取り出すときに繋ぎかえる
- 計算量
- 実装時間
    - Iterative
      - 30分
    - Recursion
      - 30分ずつ
    - Stack
      - 10分 

### 実装方針-Iterative
- 時間計算量: O(N)
- 空間計算量: O(1)
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        if (head == null || head.next == null) {
            return head;
        }

        ListNode previous = head;
        ListNode current = head.next;
        ListNode next = head.next.next;
        previous.next = null;

        while (next != null) {
            current.next = previous;
            previous = current;
            current = next;
            next = next.next;
        }

        current.next = previous;
        return current;
    }
}
```

### 実装方針Recursion1
- 方針詳細
  - 再帰(帰りながら)ポインタ張り替えをする方針
  - 「一番最後まで行ったら戻りながら2ノード間の矢印を貼り替えてね」が理解しやすく、素直なのではと思う
  - 時間計算量: O(N)
  - 空間計算量: O(N) --> スタックにO(N)積まれるから
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
  public ListNode reverseList(ListNode head) {
    return reverseListHelper(null, head);
  }

  private ListNode reverseListHelper(ListNode previous, ListNode current) {
    if (current == null) {
      return previous;
    }
    ListNode reverseNewHead = reverseListHelper(current, current.next);
    current.next = previous;
    return reverseNewHead;
  }
}
```

### 実装方針Recursion2
- 方針詳細
  - 再帰(行きながら)ポインタ張り替えをする方針
  - Iterative に似ているが、やはり考えないといけないことが多い(ポインタの移動と張り替えを同時に行う必要がある)
  - 上記の帰りがけの方が格段に素直でシンプルなので、帰りで納得したら行きがパッと実装できなくなっていた
  - current.next を previous に更新する前に、current.next を保存しておかないといけない点をエラー吐かせて気が付くを繰り返してしまった
  - 時間計算量: O(N)
  - 空間計算量: O(N)
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        return reverseListHelper(null, head);
    }

    private ListNode reverseListHelper(ListNode previous, ListNode current) {
          if (current == null) {
              return previous;
          }
          ListNode next = current.next;
          current.next = previous;
          return reverseListHelper(current, next);
    }
}
```

## Stack
- 方針詳細
  - 時間計算量: O(N) 
  - 空間計算量: O(N)
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        Deque<ListNode> stack = new ArrayDeque<>();
        ListNode node = head;
        while (node != null) {
            stack.push(node);
            node = node.next;
        }
        if (stack.isEmpty()) {
            return null;
        }
        ListNode newHead = stack.peek();
        ListNode tail = stack.pop();
        while (!stack.isEmpty()) {
            ListNode nextNode = stack.pop();
            tail.next = nextNode;
            tail = nextNode;
        }
        tail.next = null;
        return newHead;
    }
}
```
